## Key Functionalities and Capabilities

### High-Level Reasoning & Performance
- Optimized for complex, assistant-like chat tasks
- Outperforms many open-source and closed chat models

### Multilingual Support
- Native support for 8 languages:  
  English, German, French, Italian, Portuguese, Hindi, Spanish, and Thai

### Tool-Use/Function Calling
- Instruct-tuned for external tool interaction
- Enables agentic workflows

### Large Context Window
- Supports 128K token context window
- Processes long documents and extended conversations

### Open-Weights/Local Deployment
- Secure offline execution via Ollama
- Ensures full data privacy

## Ideal Use Cases

- **Complex Conversational AI**  
  Advanced chatbots requiring high understanding and reasoning

- **Multilingual Content Generation**  
  High-quality translation and generation across supported languages

- **Synthetic Data Generation**  
  Trains smaller, specialized models using its high intelligence

- **Agentic Frameworks**  
  Applications requiring tool use and multi-step decision making

## Technical Specifications

- **Architecture**: Auto-regressive transformer with Grouped-Query Attention (GQA)
- **Parameters**: 70 Billion (Dense model)
- **Data Freshness**: Knowledge cutoff December 2023
- **Ollama Formats**:  
  Available in quantized versions (e.g., Q4_K_M, Q5_0)  
  Runs on hardware from workstations to server GPUs
