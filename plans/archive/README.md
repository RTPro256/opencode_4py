## Purpose

These plans are kept for:

1. **Historical Reference** - Understanding past decisions and rationale
2. **Onboarding** - New contributors can understand project evolution
3. **Lessons Learned** - Avoid repeating past mistakes
4. **Audit Trail** - Track project development history

---

## Archive Criteria

Plans are moved to archive when:

- All planned tasks are completed
- The plan is superseded by a newer version
- The plan is no longer actively maintained
- The plan serves only as historical reference
